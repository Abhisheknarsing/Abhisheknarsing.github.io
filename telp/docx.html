Learning telp (నేర్చుకుందాం)

Welcome to the official documentation for telp, a programming language designed to be written in Telugu.

Installation Setup

Getting started with telp is easy. We provide pre-built binaries for Windows, macOS, and Linux that include everything you need, including the Node.js runtime.

1. Download

Go to the download section and choose the package for your operating system.

2. Extract

Unzip the downloaded file to a location of your choice (e.g., C:\telp on Windows or ~/telp on macOS/Linux).

3. Verify

Open your terminal or command prompt, navigate to the folder, and run:

Terminal
$ ./telp --help

Running Programs

To run a Telugu program, create a file with the .tl extension.

hello.tl
మాట్లాడు("నమస్కారం!")

Then execute it using the runner:

Terminal
$ ./telp run hello.tl

Variables (సృష్టించు)

Use the సృష్టించు keyword to declare variables. telp supports numbers, strings, arrays, and booleans.

# Creating variables సృష్టించు పేరు = "రాజు" సృష్టించు వయస్సు = 25 సృష్టించు విద్యార్థి = నిజం

Input & Output

Printing (మాట్లాడు)

Print values to the console using మాట్లాడు.

మాట్లాడు("హలో") మాట్లాడు(10 + 20)

Input (అడుగు)

Take user input using అడుగు(). It pauses execution until the user types something.

సృష్టించు పేరు = అడుగు() మాట్లాడు("హలో " + పేరు)

Control Flow

If / Else (ఉంటే / లేకపోతే)

Make decisions in your code.

ఉంటే (వయస్సు >= 18) { మాట్లాడు("మీరు ఓటు వేయవచ్చు") } లేకపోతే { మాట్లాడు("మీరు చిన్నవారు") }

Loops (కొరకు / చేయి)

Repeat code blocks. Use కొరకు for counting and చేయి for conditions.

# For Loop కొరకు (సృష్టించు i = 0; i < 5; i = i + 1) { మాట్లాడు(i) } # While Loop చేయి (x > 0) { మాట్లాడు(x) x = x - 1 }

Functions (పని)

Define reusable blocks of code with పని and return values with చల్లు.

పని కూడిక(a, b) { చల్లు a + b } సృష్టించు ఫలితం = కూడిక(10, 20) మాట్లాడు(ఫలితం)
